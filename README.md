# MonET

MonET(자본주E.T)은 청소년 사용자가 게임 플레이를 통해 금융 판단을 반복적으로 경험하도록 설계한 웹 기반 에듀테인먼트 프로젝트입니다.

## 기획 한 줄 소개

- 러닝 액션 플레이와 금융 퀴즈, 자산 선택 경험을 결합한 웹 게임
- 재미를 통해 금융 개념이 아니라 금융 판단 감각을 익히는 것을 목표로 함
- 무설치 웹 서비스로 빠른 진입과 반복 플레이를 지향함

## 브랜치 운영

- `master`: 프로젝트 개요와 문서용 브랜치
- `infra/develop`: 인프라 및 배포 기준 브랜치
- 그 외 구현 작업은 각 팀 브랜치에서 진행
